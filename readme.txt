Win Sound.exe アプリケーション
readme.txt 説明書
=*=*=*=*=*=*=*=*=*=*=*=*=*=*=*=*=*=*=*=*=*=*=*=*=*=*=*=*=*=*=*=*=*=*=*=*=
このソフトはウィンドウズのシステムサウンドで作曲をできます。
[情報][待つ120]などのコマンドを組み合わせて作ります。
コンボボックスで選ぶか直接入力して[追加]を押したら追加されます。
種類は以下のとおりです。

情報
情報のサウンドを鳴らします。

警告
警告のサウンドを鳴らします。

注意
注意のサウンドを鳴らします。

質問
質問のサウンドを鳴らします。ほとんどのPCで音の設定はありません。
コントロールパネルで変更してください。

ビープ
一般の警告音を鳴らします。

待つ〜
指定したミリ秒待ちます。
〜に数字を指定します。

ホームページに掲載します。
